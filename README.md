## Write Content
![Get used to Hugo](https://gohugo.io/getting-started/quick-start) and then:
 1) `hugo new posts/<articleTitle>.md` (as file name without blanks, e.g. `hello-world.md`)
 2) edit content
 3) `hugo server -D`

### Edit page header
